Using GeoNames APIs

API Name API Description
Oceans Enter longitude and latitude coordinates to find the name of the sea or ocean at that point (if there is one).

Weather Station Observation Enter longitude and latitude coordinates to find the nearest weather station and its latest observations.

Elevation Information Enter the longitude and latitude coordinates to find its elevation in meters as recorded by the Shuttle Radar Topography Mission.
